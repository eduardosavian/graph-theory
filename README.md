## Run

### Windows

```
.\graph.exe
```

### Linux
Em sistemas UNIX:
```
$ ./graph
```

O executável `graph` foi estaticamente compilado para Linux x86_64. O único
arquivo que necessita ser compilado é `graph.cpp`. Apenas um compilador de
C++20 é necessário (GCC/Clang/MSVC).
